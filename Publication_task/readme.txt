 1)выгрузить id авторов публикаций с удаленного компьютера, подключившись по SSH
	curl 'http://elibrary.ru/projects/API-NEB/API_NEB.aspx?ucode=7e3ea01c-d525-442a-8ec4-0394a060c197&sid=013&orgid=639'

 2)выгрузить через удаленный компьютер id публикаций для каждого автора:
	Чтобы получить номера статей автора authorid=1044837:
	curl 'http://elibrary.ru/projects/API-NEB/API_NEB.aspx?ucode=7e3ea01c-d525-442a-8ec4-0394a060c197&sid=011&authorid=1044837'

3)Со своего компьютера получать аннотации для каждой статьи (https://elibrary.ru/item.asp?id=42676777 для статьи номер 42676777):
